# Разработчик 1: Hero-секция и интерактивный глобус

## Обзор
Ваша главная задача — разработка и полировка верхней части лендинга (Hero-секции), которая включает в себя сложную трехэтапную анимацию по скроллу и интерактивный 3D-глобус (библиотека Cobe).

## Основные зоны ответственности
1. **Компонент Hero (`Hero.tsx`)**
   - Управление логикой скролл-анимации (Фаза 1: Заголовок исчезает, глобус поднимается; Фаза 2: Пауза; Фаза 3: Появление CTA-блока и сдвиг глобуса).
   - Адаптивность: корректная стилизация секции на мобильных устройствах, планшетах и десктопах (обработка `window.innerHeight / innerWidth`).
   
2. **Интерактивный глобус**
   - Настройка взаимодействия с глобусом (вращение мышью) без перекрытия прозрачными слоями (использование `pointer-events-none` у вышестоящих слоев).
   - Оптимизация производительности WebGL-канваса, чтобы избежать падения FPS при прокрутке страницы.

3. **Типографика и CTA-кнопки (Call to Action)**
   - Имплементация всех состояний кнопок (`hover`, `active`).
   - Добавление микро-интеракций, теней с фирменным фиолетовым оттенком (`#783FDD`) и анимаций появления для блока со статистикой.

## Технические детали
- Для анимаций по скроллу использовать прямое изменение DOM-свойств стилей через `useRef` и `requestAnimationFrame`. Избегать вызовов setState/useState, привязанных к `onScroll`.
- Ваш блок (Hero) является корневой "точкой входа", компоненты остальных разработчиков будут отрендерены ниже.
