# Разработчик 2: Бегущая строка и Step-by-Step Workflow

## Обзор
Ваша задача — создание средних секций лендинга: плавающей бесконечной ленты с логотипами клиентов/партнеров (Marquee) и интерактивного пошагового объяснения процесса работы Phantom Pipeline (Workflow / Timeline).

## Основные зоны ответственности
1. **Плавающая лента с клиентами (Infinite Marquee)**
   - Создание бесконечной, плавной горизонтальной прокрутки.
   - Эффект должен работать без "прыжков" при зацикливании в самом конце.
   - Добавить остановку анимации при наведении мыши (Pause on hover) для удобства изучения/рассмотрения.
   - Адаптация скорости и размера элементов под мобильные устройства.

2. **Step-by-step Workflow (Пошаговый процесс)**
   - Верстка секции, рассказывающей алгоритм "Auto-SDR" (от поиска базы до отправки письма).
   - Реализация scroll-triggered анимации, при которой карточки шагов появляются по мере прокрутки вниз.
   - Создание визуальных связей (например, соединительные линии, прогресс-бары или SVG-пути) между карточками шагов.

3. **Стилизация и производительность**
   - Использование Tailwind CSS для UI.
   - Строгое соблюдение дизайн-системы (глубокий фиолетовый `#783FDD`, акценты, серые тексты `#6B7280`).
   - Для scroll-анимаций рекомендуется использовать легковесные CSS-анимации, Intersection Observer либо Framer Motion (согласовать с командой).

## Интеграция
- Ваши компоненты будут вставлены сразу под `Hero.tsx` в главном файле (`Index.tsx` или `Landing.tsx`).
- Необходимо синхронизироваться с Разработчиком 1 для обеспечения консистентных отступов (`padding/margin`) между секциями на разных разрешениях экрана.
